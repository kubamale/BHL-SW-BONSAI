# MartianCalendarFrontend

Frontend for the Martian Calendar, built with React, Vite, and TypeScript.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Access the app at:
   ```
   http://localhost:5173/
   ```

## Build for Production
```bash
npm run build
```
